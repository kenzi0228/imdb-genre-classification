#  IMDB Genre Prediction - Naive Bayes

##  Project Overview
This project aims to predict the genre of a movie using numerical features such as:
- Number of votes
- IMDB rating
- Meta score

A Naive Bayes model is used for classification.

## Methodology
1. Data preprocessing
2. Feature selection
3. Genre encoding
4. Train/test split
5. Model training (GaussianNB)
6. Evaluation (accuracy, precision, recall, F1-score)

## 📊 Results
The model provides baseline performance but is limited by the simplicity of the features.
